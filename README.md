## Installation

* $ gem install civicrm

## Getting started

* CiviCrm.api_key = 'YOUR_API_KEY'
* CiviCrm.api_base = 'https://www.example.org/path/to/civi/codebase/'
* CiviCrm.authenticate('demo', 'demo')

## CiviCrm Objects

* CiviCrm::Contact.all    # get list of contacts
